In your AndroidManifest.xml, find the <application> tag and add:

  android:networkSecurityConfig="@xml/network_security_config"
  android:usesCleartextTraffic="true"

So it looks like:

  <application
      android:name="io.ionic.starter.MyApp"
      android:allowBackup="true"
      android:icon="@mipmap/ic_launcher"
      android:label="@string/app_name"
      android:networkSecurityConfig="@xml/network_security_config"
      android:usesCleartextTraffic="true"
      android:roundIcon="@mipmap/ic_launcher_round"
      android:supportsRtl="true"
      android:theme="@style/AppTheme">

Also make sure you have INTERNET permission in the manifest:
  <uses-permission android:name="android.permission.INTERNET" />

File location:
  android/app/src/main/AndroidManifest.xml
  android/app/src/main/res/xml/network_security_config.xml  ← create this folder + file
